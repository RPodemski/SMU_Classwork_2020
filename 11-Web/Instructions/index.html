Latitude vs X

The purpose of this project was to analyze how weather changes as you get closer to the equator. We first pulled data from the OpenWeatherAPI to create a dataset on more than 500 citites.

After creating the dataset, we used Matplotlib to plot the variables of the weather versus latitude. The variables that we looked at were: temperature, cloudiness, wind speed, and humidity. This website provides the source data used along with the visualizaitons created as part of the analysis. There are also observations on trends and correlations observed.
